Button 按钮 ​

基础用法 ​

使用 type，round,circle来定义按钮样式

禁用状态 ​

使用 disabled 属性来定义按钮是否被禁用

图标按钮 ​

通过添加 icon 属性给按钮添加图标，也可以通过插槽，使用 icon 组件将图标插入

调整尺寸 ​

通过 sze 属性给按钮调整尺寸，有 large，medium，small 选项,medium 是默认尺寸

按钮组 ​

使用<yl-button-group>对多个按钮分组

加载状态的按钮 ​

设置 loading=true，显示加载状态
